Unsupported browser

Transcrobes has been tested and works best on either Google Chrome, Microsoft Edge, Opera and other Chromium-based browsers on either the desktop (Windows, Linux and Mac) or on Android.

Unfortunately, other browsers (including everything on iOS) don't properly support all the latest web standards, so do not work properly. We are working hard on bringing compatibility to all platforms. In the meantime, please use one of the supported browsers. Thank you.
